How it works.

Our API.

Our (state-of-the-art thank you very much) API uses the latest code on the market. The Flamingo bot will send the info it finds (humidity, weather + changes, ground surface, grass, etc) to our cloud servers who will then transmit all the info you have to either you web-based account or to the application you have installed on your phone. The application will display all transmitted information which you then can see.
